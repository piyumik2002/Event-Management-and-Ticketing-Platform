# 🎟️ TikXpress - Digital Event Management & Ticketing Platform

TikXpress is a modern, premium full-stack digital event management and ticketing ecosystem designed to bridge the gap between event organizers and audiences in Sri Lanka. It provides a transparent, reliable, and secure solution for discovering and booking live events and movies.

## ✨ Features
* **Live Event Discovery:** Filter and search events smoothly by category, venue, or keywords.
* **Interactive Seat Booking:** Premium real-time seat selection dashboard for a hassle-free checkout experience.
* **Responsive Dark UI:** Crafted beautifully with React, Tailwind CSS, and glassmorphism design trends.
* **Secure Checkout:** Ticket pricing and summary configurations built dynamically.

## 🛠️ Tech Stack
* **Frontend:** React (Vite), Tailwind CSS, Lucide React
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
